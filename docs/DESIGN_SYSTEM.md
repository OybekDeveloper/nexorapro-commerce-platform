# Design system

## Brand direction

nexorapro.uz uses a focused, premium commerce visual language. The primary brand color is `#10a184`; neutral surfaces and restrained accent usage keep dense retail data readable without making the interface visually noisy.

## Foundations

- Primary: `#10a184`
- Backgrounds: white and soft neutral surfaces
- Typography: high-contrast hierarchy with compact admin labels
- Radius: rounded controls and cards with consistent proportions
- Motion: short, purposeful transitions with reduced-motion support
- Icons: custom nexorapro.uz SVG marks supported by Lucide interaction icons

## Interface principles

1. Keep storefront content product-led and editorial.
2. Keep admin workflows data-dense but easy to scan.
3. Use color to communicate actions and state, not decoration.
4. Preserve keyboard focus, explicit labels, and WCAG AA contrast.
5. Provide mobile alternatives for wide tables and operational panels.

## Core experiences

- Premium customer storefront
- Executive commerce dashboard
- Product catalog and publishing controls
- Point-of-sale workspace
- Inventory, order, localization, and analytics modules
